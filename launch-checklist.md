# Launch Checklist

Prep doc for client meeting — everything needed to take the site live.

## Decisions needed from client

- [ ] **Deposit amount** — dollar amount for the Stripe booking deposit
- [ ] **Calendly scheduling setup** — confirm the Calendly link, event duration/buffer, and availability/time zone
- [ ] **Domain name** — exact name to purchase, and who owns/pays for the registrar account
- [ ] **Service area** — confirm ZIP codes/cities are correct (currently set to North County San Diego)
- [ ] **Pricing sign-off** — sedan vs. truck/SUV rate card, confirm final numbers per package
- [ ] **Business legal name / contact info** — for Google Business Profile, site footer, and legal fields

## Setup tasks

- [ ] Purchase domain and point DNS to hosting (Vercel)
- [ ] Create Stripe account and generate the deposit Payment Link
- [ ] Create/configure Calendly account and event type
- [ ] Set up Google Business Profile (name, service area, phone, hours, photos)
- [ ] Enter Calendly URL and Stripe deposit link into Sanity Site Settings
- [ ] Set up business email (e.g. hello@yourdomain.com)
- [ ] Confirm phone number displayed on site is correct and monitored
- [ ] Set up analytics (Google Analytics / Search Console) before launch
- [ ] Test full booking flow end-to-end (Calendly → Stripe deposit) with real links
- [ ] Decide on business insurance/liability info if it should appear on the site
- [ ] Finalize logo/branding assets in Sanity
